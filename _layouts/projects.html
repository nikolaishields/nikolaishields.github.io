[In Progress] A simple and concise operating system and webapp for mining cryptocurrencies and managing multiple mining rigs. conveniently allows for user to track metrics and profits, with a fully accessible API. Created using MongoDB, Polymer, and some good old fashioned Linux knowledge.


RFID Door Lock

A door lock built for a the local makerspace in town, built on a raspberry pi. The doorlock interfaces with slack to allow for the convenient monitoring of shop activities. Won't go into to much detail as its still in use today, if you want to learn more or see it in action and are in the Pensacola area, head down to Emerald Coast Communnity of Makers.


Gitspotlight

Is an interactive application that makes reccomendations to software engineering students looking for opensource projects to work on, using a familiar "swipe" interface. Built for a class in college.
